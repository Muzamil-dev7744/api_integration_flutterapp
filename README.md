# API Integration Flutter App

This is a Flutter application that demonstrates **API integration** and dynamic data handling. The app fetches data from an external API and displays it in a clean and user-friendly interface.

---

## 🚀 Features

* 🔗 API Integration using HTTP requests
* 📱 Clean and responsive UI
* 🔄 Dynamic data fetching
* ⚡ Fast performance with Flutter
* 🧩 Easy-to-understand project structure

---

## 🛠️ Technologies Used

* Flutter (Dart)
* HTTP Package
* REST APIs

---

## 📂 Project Structure

```
lib/
│── models/        # Data models
│── services/      # API services
│── screens/       # UI screens
│── widgets/       # Reusable widgets
│── main.dart      # Entry point
```

---

## ⚙️ Getting Started

Follow these steps to run the project locally:

### 1. Clone the repository

```
git clone https://github.com/YOUR_USERNAME/api_integration_flutterapp.git
```

### 2. Navigate to project folder

```
cd api_integration_flutterapp
```

### 3. Get dependencies

```
flutter pub get
```

### 4. Run the app

```
flutter run
```
